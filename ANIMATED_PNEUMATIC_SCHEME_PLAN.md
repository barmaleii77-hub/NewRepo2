# ?? АНИМИРОВАННАЯ СХЕМА ПНЕВМАТИЧЕСКОЙ ПОДВЕСКИ

## ?? ВОЗВРАЩЕНИЕ К ОСНОВНОЙ ЦЕЛИ

**Цель проекта:** Создание интерактивной анимированной схемы пневматической подвески с:

1. **Визуализацией компонентов:**
   - Пневматические цилиндры (4 штуки)
   - Клапаны управления
   - Воздушные магистрали
   - Ресивер (воздушный резервуар)
   - Компрессор

2. **Анимацией работы:**
   - Движение поршней в цилиндрах
   - Потоки воздуха по магистралям
   - Открытие/закрытие клапанов
   - Изменение давления (цветовая индикация)

3. **Интерактивностью:**
   - Управление через UI панели
   - Изменение параметров в реальном времени
   - Графики давления и положения
